# Queuing-System-Simulation

Simulation and probabilistic analysis of two servers in series 
applied to a queuing model.

## Overview
Modeling and analysis of a queuing system with two sequential 
servers, including performance metrics such as waiting times, 
throughput, and server utilization.

## Technologies
- Python
- NumPy
- SciPy
- MatPlotLib

## Key Concepts
- Queuing theory
- Discrete event simulation
- Descriptive statistics
- Probability distributions
